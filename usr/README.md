# Software Engineering Case Studies

A modern Flutter application designed to showcase software engineering case studies, architectures, and technical achievements. The app presents a clean, readable interface for exploring complex technical challenges and their solutions.

## Features

- **Case Study Listings**: Browse through summarized engineering challenges with key technologies highlighted.
- **Detailed Case Views**: Dive deep into specific case studies, including the initial challenge, the applied solution, and measurable key results.
- **Technology Tags**: Quickly identify the tech stack used in each case study via stylized tags.
- **Beautiful UI/UX**: Implements a modern Material 3 design with smooth Hero animations and responsive layouts.
- **Light & Dark Mode Support**: Adaptive theming based on system preferences.

## App Structure

- **Models**: Defines the data structure for case studies (`CaseStudy`).
- **Data**: Contains comprehensive mock data for real-world engineering scenarios.
- **Screens**: 
  - `HomeScreen`: Displays the list of all available case studies.
  - `CaseStudyDetailScreen`: Shows in-depth information about a specific case study.
- **Widgets**: Reusable components like `CaseStudyCard` and `TagChip`.
- **Theme**: Custom Material 3 theme configuration (`AppTheme`).

## Tech Stack

- **Framework**: Flutter
- **Language**: Dart
- **Design System**: Material Design 3

## Setup & Run

1. Ensure you have Flutter installed and configured.
2. Clone the repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run the app using `flutter run` on your preferred device or emulator.
